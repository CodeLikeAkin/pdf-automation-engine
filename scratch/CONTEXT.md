# Scratch Context: Temporary Workspace

In this room, we conduct experiments and test one-off fixes. This is the "Lab" of the project.

## What belongs here?
- **Temporary Scripts**: Test snippets for new logic (e.g., regex testing).
- **Draft Experiments**: Fragments of chapters that aren't ready for the `/drafts` folder.
- **Malformed Data**: Files used to test validation and error handling (e.g., `test_malformed.md`).

## Rules
1. **No Permanence**: Do not store project-critical code or content here.
2. **Periodic Cleaning**: This folder should be cleared out after successful project milestones.
3. **Naming**: Use descriptive names like `test_font_caching.py` to keep the mess manageable.
