# Skill: UI/UX Pro Max
**Objective**: Apply professional design systems, typography rules, and UX patterns to digital products/PDFs.

## Reasoning Rules
1. **The 8pt Grid**: Ensure all spacing (margins, padding, gaps) is a multiple of 8px.
2. **Visual Hierarchy**: 
   - H1: Bold, Large (24pt+), High Contrast.
   - Body: High readability (10-12pt), 1.6 Line Height.
   - Accents: Use subtle grays or brand-specific primary colors.
3. **Typography Pairings**: Always pair a "Personality" font (Headers) with a "Utility" font (Body).
4. **Information Density**: If a page is too "dense," automatically suggest breaking it into two pages or using call-out boxes.
5. **Accessibility**: Ensure a minimum contrast ratio of 4.5:1.

## Implementation Tasks
- Generate CSS variables for the `/design` room.
- Audit the final PDF layout for "anti-patterns" (e.g., text too close to the edge).
